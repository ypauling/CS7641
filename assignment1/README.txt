Datasets:
1. Hand digit recognition.
2. Breast cancer wisconsin dataset.

Packages used:
python (3.6.9)
scikit-learn (version 0.21.2)
numpy (1.16.5)
matplotlib (3.1.1)

Folder structure assumed:
>breast_cancer_analysis.py <file>
>digit_analysis.py         <file>
>breast_cancer/            <folder>
    >figures/              <folder>
>digits_recognition/       <folder>
    >figures/              <folder>

Simply run *.py files, and the corresponding figures will be generated automatically.  All the necessary information will be printed in the terminal.
